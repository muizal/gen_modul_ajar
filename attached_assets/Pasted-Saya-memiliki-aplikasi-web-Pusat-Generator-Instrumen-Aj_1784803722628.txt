# TUJUAN

Tahap ini bertujuan mempersiapkan aplikasi agar:

1. Khusus untuk jenjang SMA.
2. Kelas hanya X, XI, XII.
3. CP resmi menjadi sumber utama dokumen.
4. Sistem tidak pernah membuat CP palsu/fallback.
5. Data CP dipisahkan dari `index.html`.
6. Struktur siap dikembangkan menjadi generator dokumen berbasis AI pada tahap berikutnya.

Jika data tidak tersedia, return array kosong.

# 5. PISAHKAN DATA CP DARI INDEX.HTML

# 7. BEDAKAN CP DAN PEMETAAN MATERI

CP adalah kompetensi resmi berdasarkan Fase.

Sedangkan:

* TP
* materi
* topik
* JP
* jumlah pertemuan

Simpan pemetaan tersebut dalam:

{

}

Dengan demikian CP Fase F dapat digunakan oleh kelas XI dan XII tetapi materi keduanya dapat berbeda.

# 8. BUAT HELPER CP

Buat fungsi:

getCPData(mapel, fase)

Fungsi hanya mengambil CP yang tersedia.

Tidak boleh membuat fallback.

Buat:

hasCPData(mapel, fase)

Return boolean.

Contoh konsep:

function hasCPData(mapel, fase) {

```
const data = getCPData(mapel, fase);

return !!(
    data &&
    data.cp &&
    data.cp.trim() !== ""
);
```

}

Silakan sesuaikan implementasi dengan kode existing.
